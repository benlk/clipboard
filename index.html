Clipboard CSS

This is a utility stylesheet for throwing up fast, one-page, mostly-text sites. It's not meant to be used for developing apps or providing structure or anything.

Using it is as simple as downloading clipboard.css, using it in your project and citing where you found it. clipboard.css is licensed under the MIT License, but the Buttons library included here is used under the Apache License v 2.0.

Inspiration for this came from Uberwriter.
